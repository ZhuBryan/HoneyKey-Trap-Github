# ACME Internal Dashboard

Internal dashboard for managing ACME Corporation's project metrics and team analytics.

> **Note**: This is an internal tool. Please contact DevOps for access credentials.

## Features

- Real-time project metrics visualization
- Team performance analytics
- Integration with ACME Internal API
- Automated reporting

## Quick Start

```bash
# Install dependencies
npm install

# Configure environment (see config.js)
# Run development server
npm run dev
```

## Configuration

Copy `config.example.js` to `config.js` and update with your credentials.

See `src/config.js` for current configuration.

## API Integration

This dashboard connects to ACME's internal API for data retrieval. Ensure you have valid API credentials before running.

## Development

```bash
npm run dev      # Start dev server
npm run build    # Production build
npm run test     # Run tests
```

## Deployment

Contact the infrastructure team for deployment to internal servers.
